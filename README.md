## futaba new res counter
このUserscriptはChromeアドオン[ふたクロ](http://futakuro.com/)でふたば☆ちゃんねるのスレを更新したときに新着レス数およびスレ消滅を取得しタブに表示します。  

[Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)を先にインスールしてからスクリプトをインストールして下さい。  

※このUserscriptは[ふたクロ](http://futakuro.com/)アドオンの機能を使用しているためChrome専用となります。  
※その他のUserscriptは[こちら](https://github.com/akoya-tomo/futaba_auto_reloader_K/wiki)の一覧からどうぞ

## 機能
* 新着レス数およびスレ消滅状態をタブのタイトルに表示  
 他のタブを見ていてもLIVE状態のスレに新着レスがあるかスレが消えたかが分かります。  
 ページ末尾でホイールダウンするとリセットできます。  
* 板名をタブに表示できます。（デフォルト：有効）

## 設定
機能の動作はスクリプト冒頭の大文字変数をエディタで編集すれば変更することができます。  

* USE\_BOARD\_NAME :タブに板名を表示する  
  - タブに板名（二次裏のみサーバー名）を表示します。  

## 注意事項
※このUserscriptはChromeで[ふたクロ](http://futakuro.com/)アドオンをインストールしている事が前提となります  
※将来Firefox用ふたクロがリリースされた場合、本スクリプトはこのままで動作すると思いますが、現時点では動作未確認です。
